Blockchains

This directory contains the code which calculates network emissions for different blockchains.
Each sub-directory contains the code relevant to a single blockchain, with the exception of common which contains shared code used in calculating emissions for multiple blockchains.

Documentation template

The README-template.md file is the documentation template to use whenever a new blockchain is added to this directory. This is to keep the documentation of the different blockchains consistent and ensure that all the important facts are included whenever a new cryptocurrency is added.
e.g. if you add a newcoin sub-directory, please copy README-template.md to newcoin/README.md and edit the content accordingly.
